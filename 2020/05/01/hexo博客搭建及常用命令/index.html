hexo博客搭建过程

  • 本文主要介绍如何从0搭建自己的个人博客，并部署到github, 包括博客主题，及常用设置，常用命令。

1、确保安装 node.js、npm、git

  • 在cmd命令提示符下，依次验证node -v，确保是在全局变量下，如无法获取版本信息，请配置环境变量

  • 安装cnpm,目的是安装其它插件快一点,可先cnpm -v查看是否已安装

  • 全局安装hexo框架

  • 在想放的位置建文件夹，如：blog, cd到该文件下执行

    安装完毕，执行hexo s, 然后访问local host：4000即可查看

2、更换好看的博客主题

  • 我的是经典的next主题

  • 首先下载主题请百度

  • blog文件夹下的_config.yml找到theme: 更换为下载的主题名

  • cd 到blog下，执行 hexo clean 、hexo g、 hexo s

  • 基于主题的美化和细节设置参考其它文章

3、 推送到github及hexo常用命令

1、博客只能在本地浏览是没意思的，我们可以推送到github去绑定。

2、新建仓库，名字严格要求: github名字.github.io,比如我的 wangxueleli.github.io

3、别忘了blog文件夹下的_config.yml，最后一行repo换成github新建项目 的地址

4、推送到github, cd 到blog下，执行 hexo d 这样都可以远程访问了
